# Requirements
    cmake

# Build
```bash
    git clone https://github.com/smtyksl/chess-threat.git
    cd chess-threat
    mkdir build
    cd build
    cmake ../
    make
    
```
# Run
* sample
```bash
    ./chess-threat ../files/board1.txt
```
* Note: you can check results with sonuclar.txt inside of files directory.
